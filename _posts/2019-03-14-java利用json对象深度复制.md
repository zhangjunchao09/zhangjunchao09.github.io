# 对象拷贝

利用net.sf.json将对象转成json字符串，再利用net.sf.json将字符串转成对象

## 单个对象深度拷贝

```
/**
 * 通过json复制对象属性
 */
public class CopyUtils{

    private static ObjectMapper readMapper = new ObjectMapper();
    private static ObjectMapper writeMapper = new ObjectMapper();

    static {
        readMapper.configure(DeserializationFeature.FAIL_ON_UNKNOWN_PROPERTIES, false);
    }

    /**
     * 利用json工具完成对象深度复制
     * @param s
     * @param c
     * @param <T>
     * @return
     */
    public static <T> Object jsonCopyPropertys(Object s, Class<T> c) {
        String sStr;
        try {
            sStr = writeMapper.writeValueAsString(s);
        } catch (JsonProcessingException e) {
            logUtils.error(e);
            throw new InnerMessageException(ResponseCode.DATA_ERROR, "对象转Json错误");
        }
        T t;
        try {
            t = readMapper.readValue(sStr, c);
        } catch (IOException e) {
            logUtils.error(e);
            throw new InnerMessageException(ResponseCode.DATA_ERROR, "Json转对象错误");
        }
        return t;
    }
}

```

## 对象集合深度拷贝

```
/**
 * 通过json复制对象属性
 */
public class CopyUtils{

    private static ObjectMapper readMapper = new ObjectMapper();
    private static ObjectMapper writeMapper = new ObjectMapper();

    static {
        readMapper.configure(DeserializationFeature.FAIL_ON_UNKNOWN_PROPERTIES, false);
    }
    /**
     * 利用json工具完成对象list深度复制
     * @param ss
     * @param clz
     * @param <T>
     * @return
     */
    public static <T> List<T> jsonCopyListPropertys(List<? extends Object> ss, Class<T> clz) {
        String sStr;
        try {
            sStr = writeMapper.writeValueAsString(ss);
        } catch (JsonProcessingException e) {
            logUtils.error(e);
            throw new InnerMessageException(ResponseCode.DATA_ERROR, "对象转Json错误");
        }
        List<T> ts = new ArrayList<>();
        try {
            ts = readMapper.readValue(sStr, new TypeReference<List<T>>() {
            });
        } catch (IOException e) {
            e.printStackTrace();
        }
        return ts;

    }
}
```
